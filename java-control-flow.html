Part 1: Decision Making

Programs are not static; they need to make choices based on data. In Java, we use If-Else for complex logic and Switch for specific value matching.

1

If-Else Statements

The most fundamental decision-making tool. Evaluates a boolean expression (True or False).

Logic
"If condition A is true, do this. Otherwise, do that."
Best Used For
Ranges (e.g., score > 50), boolean checks, complex logic operators
if else else if && ||
The Temperature Check
int temperature = 30; if (temperature > 30) { // This runs if temp is 31 or higher System.out.println("It's hot outside! Wear shorts."); } else if (temperature >= 20) { // This runs if temp is between 20 and 30 System.out.println("It's nice. Wear a t-shirt."); } else { // This runs if none of the above are true (temp < 20) System.out.println("It's cold. Bring a jacket."); }
2

Switch Statements

A cleaner alternative to if-else when comparing a single variable against a list of known values.

Logic
"Check this variable. Does it equal Case 1? Case 2? Case 3?"
Best Used For
Menus, days of the week, specific ID codes
switch case break default
Critical Keywords Explained
case: The value we are checking against.
break: Stops the code from "falling through" to the next case.
default: Runs if no cases match (like the else block).
The Vending Machine
int selection = 2; switch (selection) { case 1: System.out.println("Dispensing: Coke"); break; case 2: System.out.println("Dispensing: Pepsi"); break; // vital to stop execution here! case 3: System.out.println("Dispensing: Water"); break; default: System.out.println("Invalid Selection"); }

Part 2: Iteration (Loops)

Loops allow you to repeat a block of code multiple times without rewriting it.

3

For Loops

The standard loop when you know exactly how many times you want to iterate.

Logic
"Start at X; Keep going while Y is true; Change X by Z every time."
For Loop Structure
for (
Initialization
;
Condition
;
Update
)
for i++ i-- i += n
The Countdown
// Logic breakdown: // 1. int i = 10 (Start at 10) // 2. i > 0 (Run as long as i is positive) // 3. i-- (Decrease i by 1 after every loop) for (int i = 10; i > 0; i--) { System.out.println("T-minus: " + i); } System.out.println("Blast off!");
4

While Loops

Used when the number of iterations is unknown. It relies on a state or condition.

Logic
"Keep doing this while the condition is true."
Warning: You must manually update the variable inside the loop, or you will create an Infinite Loop that crashes your program.
while do-while break continue
Loading a File
boolean fileIsLoading = true; int percentLoaded = 0; while (fileIsLoading) { System.out.println("Loading... " + percentLoaded + "%"); percentLoaded += 25; // Simulate progress // Check if finished if (percentLoaded >= 100) { fileIsLoading = false; // This stops the loop! } } System.out.println("Download Complete.");
